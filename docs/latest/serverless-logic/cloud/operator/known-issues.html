Serverless Logic Operator Known Issues, Limitations and Roadmap

The Serverless Logic Operator is currently in Alpha version, is under active development.

  • Roadmap
    • CNCF Specification v0.8 Alignment
    • Workflow Productization Profile
    • Knative Integration
    • GitOps
    • Operator SDK, OLM, OperatorHub

Roadmap

The following issues are currently being prioritized:

CNCF Specification v0.8 Alignment

  • Implement admission webhooks for workflow validation

Workflow Productization Profile

  • Enable toggle Workflow CR from devmode to production mode and vice-versa

  • Review build failures and signal the reasoning in the Events API

  • Evaluate internal registry integration on OpenShift, Kubernetes and Minikube

Knative Integration

  • SonataFlow Operator integration with Knative Eventing

  • Knative Serving Extension for Serverless Workflow specification

GitOps

  • Extend the SonataFlow Operator with Jib builder

  • Add external built image integrity validation

Operator SDK, OLM, OperatorHub

  • Enable SonataFlow Operator for level 2 - Seamless Upgrades
